# Todo Directory

This directory contains detailed implementation plans for larger features.

For the main task list, see [docs/todo.md](../docs/todo.md).

## When to Add Files Here

Add a markdown file here when a feature requires:
- Detailed technical specification
- Multi-file changes
- Step-by-step implementation plan
- Architecture decisions

## Naming Convention

Use kebab-case: `feature-name.md`

## Template

```markdown
# Feature Name

## Problem
[What problem does this solve?]

## Solution
[High-level approach]

## Files to Modify
[List of files and what changes]

## Implementation Steps
[Ordered list of tasks]

## Edge Cases
[What edge cases need handling?]
```
